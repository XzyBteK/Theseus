# fffe0000 (Dashboard Title Data)

`fffe0000` is the original Xbox dashboard's title ID. Theseus uses the same ID so existing UIX Lite installations continue to read and write to the same persistent state location without migration.

The dashboard stores per-user state here at runtime: clock settings, video configuration, parental controls, soundtrack metadata, and similar. The folder is empty in the public repo; the dashboard creates files here as you use it.
